Welcome to MedTrack

MedTrack is a cloud-enabled healthcare system that simplifies doctor-patient communication.

  • 📅 Book Appointments Online
  • 📝 Submit & Access Diagnosis Reports
  • 🔐 Secure Access for Patients and Doctors

This project uses AWS (EC2, DynamoDB, SNS), Flask, and Python to modernize healthcare workflows.
